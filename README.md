# 🎨 Color Changer

A simple JavaScript project that allows users to change the background color of the page by either:

- Clicking a **Random Color** button
- Typing a custom color and clicking **Apply Color**

---

## 🚀 Features

- Generate a random color
- Apply custom colors manually
- Display the current active color
- Smooth UI with gradients, animations, and hover effects
- Beginner-friendly JavaScript DOM project

---

## 🛠️ Built With

- **HTML**
- **CSS**
- **JavaScript**

---

## 📸 Preview

Color Changer UI includes:

- Random Color button
- Input field for custom color
- Apply Color button
- Current Color display
